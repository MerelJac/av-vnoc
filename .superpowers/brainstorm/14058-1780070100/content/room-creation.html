A · Manual Entry Only

Users create rooms one by one via a simple form. Clean and explicit — the app's room list is fully user-controlled. No confusion from vendor naming mismatches.

B · Import from Vendor

A one-time wizard reads room names from Poly Lens (and YMCS) and creates app rooms in bulk. Devices get auto-assigned in the same step. Fast for initial setup — but vendor room names may not match what the customer calls them.

C · Smart Suggestions + Manual (Recommended)

When a device is unassigned, the app shows the vendor-side room name as a one-click suggestion ("Poly says this device is in 'Conf A' — create that room?"). Users can accept, rename, or pick an existing room. Manual creation always available too.
